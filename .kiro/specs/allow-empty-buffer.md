# Allow Empty Buffer

## Problem

KiroBuffer does not load on an empty buffer. It should load without context.

## Requirements

1. KiroBuffer should open with no file in the buffer
2. Should open even with no prompt and no file (just open Kiro)

## Acceptance Criteria

- [x] KiroBuffer should open with no file in the buffer
- [x] KiroBuffer should open with no prompt and no file (empty chat)
